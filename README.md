# 🏏 Cricket Data Analytics using Python & Power BI

## 📌 Overview
An end-to-end data analytics project that uses web scraping, Python, Pandas, and Power BI to analyze cricket player and team performance. The project covers data collection, cleaning, transformation, analysis, and dashboard creation to generate actionable insights.

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- BeautifulSoup (Web Scraping)
- Power BI
- Jupyter Notebook

## 📊 Key Features
- Scraped cricket statistics from online sources
- Cleaned and transformed raw data using Python
- Performed Exploratory Data Analysis (EDA)
- Built interactive Power BI dashboards
- Analyzed player and team performance metrics

## 📈 Dashboard Insights
- Top Performing Players
- Team Comparison Analysis
- Batting & Bowling Statistics
- Interactive KPIs and Filters
